# React-Todo-List-Express-Server
## Separate the Front-End and Back-End mini project.
### 自學將 todo-list 的前端與後端分開。
### 前端框架：React
### 後段框架：Node Express
### 資料庫：Firebase Firestore Database

### Prerequisites
該電腦需要有 Node 環境。

### Installing

git clone
