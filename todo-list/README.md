# Todo List App

## Getting Started

### Prerequisites
該電腦需要有 Node 環境。

### Installing
Downloads a package and it's dependencies.
```
npm install
```

Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
```
npm start
```
