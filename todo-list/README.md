## Getting Started

### Prerequisites
該電腦需要有 Node 環境。

### Installing
Downloads a package and it's dependencies.
```
npm install
```

Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
```
npm start
```

使用者需自行定義的地方:
在 ./src/App.tsx 內
```
// Define API
const GET_DATA = "";
const CREATE_DATA = "";
const UPDATE_DATA = "";
const DELETE_DATA = "";
const DATA_STATUS = "";
```
